Navigating the Liazon Directory

There are 3 different navigation areas in the directory:

Top Navigation

The top navigation contains section specific items for moving around the current section.

Left Navigation

The left navigation contains site-wide items for moving between and within sections.

Bottom Navigation

The bottom navigation contains miscellaneous items like the help link.
